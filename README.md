# بلوقن نشر تلقائي - Auto Publish

**المطور:** @DRR_R2  
**القناة:** @phpandpy  

نشر تلقائي للرسائل في المجموعات أو القنوات مع إعدادات مرنة.

## 🚀 طريقة الاستخدام
1. افتح إعدادات البلوقن
2. حدد الرسالة المراد نشرها تلقائياً
3. حدد الفترة الزمنية بين كل نشر

## 🔧 التثبيت
ضع الملف في مجلد `plugins` وافعله من إعدادات التطبيق.
